Portfolio

SuperMega Shuffle

SuperMega Shuffle is a project for an Independent Study course at Butler University. The purpose of this course was to learn how to use AngularJS, specifically to make one-page web applications. The application gets the YouTube channel SuperMega's uploaded videos and puts them into a playlist. You can shuffle this playlist and play the videos in the same page, with the YouTube Iframe API. This project uses HTML, CSS, and JavaScript. Libraries include Bootstrap and AngularJS.

SuperMega's Channel

Thanks to Fran Dios, whose tutorial "Make A YouTube App With AngularJS and OnsenUI" was helpful to help me learn AngularJS. Part of his code from this tutorial is used for this project, and the MIT license is included in the GitHub repository.

SITEAW Rebranding

Our four group members were given the task of creating a new web presence for the non-profit organization Shifting Ideas Through Education of African Women. After meeting with the leader of SITEAW, we decided on four areas of improvement:

  • Taking the website to Wordpress and updating the design
  • Creating new graphics for SITEAW's new media
  • Improving social media outreach and posting strategies
  • Updating printed materials and including a more focused outreach

I worked on the website design, Bram worked on the graphic design, Tessa focused on social media, and Caitlyn focused on printed materials. By meeting regularly and collaborating ideas, we were able to bring a new, sleeker look to SITEAW.
